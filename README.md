# CSS Loading Spinner Animation

Um componente de indicador de carregamento (Loader/Spinner) minimalista, elegante e altamente performático desenvolvido exclusivamente com HTML5 e CSS3. O projeto apresenta uma animação rotacional contínua de um arco circular vazado, ideal para ser integrada em estados de espera, carregamentos assíncronos (AJAX/Fetch) ou telas de transição (Preloaders).

## Tecnologias Utilizadas

* HTML5: Estruturação semântica simples e leve do contêiner do carregador.
* CSS3: Estilização completa, uso de formas circulares perfeitas e gerenciamento de animações nativas.
* Animações em CSS: Utilização de keyframes (`@keyframes`) para controlar propriedades de rotação (`transform: rotate`) de forma otimizada para o navegador.

## Funcionalidades e Técnicas Aplicadas

* Design Minimalista em Arco: Construção do spinner baseada em uma única borda circular translúcida e um arco semitransparente em branco puro, garantindo uma estética limpa, moderna e profissional.
* Movimento Rotacional Infinito: Aplicação de animações contínuas (`infinite`) com ritmo de interpolação linear (`linear`) para assegurar que o spinner gire de forma fluida, sem quebras ou travamentos visuais.
* Técnicas Avançadas de Bordas CSS: Modelagem do arco vazado utilizando a propriedade `border-radius: 50%` combinada com o isolamento de cores em propriedades específicas (como `border-top-color`), eliminando a necessidade de imagens ou arquivos SVG.
* Centralização Absoluta na Viewport: Uso estratégico de CSS Flexbox para alinhar horizontal e verticalmente o componente no centro exato da tela sobre um plano de fundo azul sólido vibrante.
